2.1. Tarfile extraction

If the package was downloaded as a tarfile, it should have the extension of tar.bz2 or tar.gz. For either case, cd into the directory in which you will install the directory tree and issue the command

for the tar.bz2 extension 

# tar -xjf <path> /os9disasm.tar.bz2

OR, for the tar.gz extension.

# tar -xzf <path> os9disasm.tar.gz

Note

The filename may vary depending upon whether you have the complete package including both OS9Disasm and g09dis or just the separate packages.

In either case, a directory named os9disasm or cocodisx.x will be created in the current location, and all files will be created under that directory.
